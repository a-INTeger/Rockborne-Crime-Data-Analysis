# Crime Analysis Project
This is a project as part of a major crime EDA project as part of Rockborne.
Github link: https://github.com/a-INTeger/Rockborne-Crime-Data-Analysis

NOTE: Data Folder is not shown in the repository <br>
**REQUIRES PYTHON 3.14.3**
## File structure
```
│   .gitignore
│   cleaning.ipynb
│   exploration.ipynb
│   Project Statement.pdf
│   Project Report.pdf
│   README.md
│
└───Data
    ├─── Clean
    │
    └─── Raw
```
### File Explanation
- `./Data/Clean` : Directory to hold output for cleaned data
- `./Data/Raw` : Directory to hold raw `.csv` or `.xlsx` files used for data analysis
- `cleaning.ipynb` : Notebook for cleaning data
- `exploration.ipynb` : Notebook for exploring data
- `README.md` : readme file for explaining the project (START HERE)
- `Project Statement.pdf` : Project statement discussing problem and project management
- `Project Report.pdf` : Project Report detailing analysis and findings (START HERE BEFORE LOOKING THROUGH NOTEBOOKS)
